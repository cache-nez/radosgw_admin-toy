## A toy program illustrating usage of boost::program_options for arguments parsing
### Usage:

$ cmake .

$ make

$ ./radosgw-admin \<cmd> [options...]


Available commands and options:

--help

user create --pid=\<pid> --display-name=\<display-name> --email=\<email>

user delete --pid=\<pid>

user info --pid=\<pid>
